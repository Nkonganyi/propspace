# PropSpace – Property Listing App

## Overview

PropSpace is a full-stack real-time property listing application where users can register, authenticate, create and manage property listings.

Users can:

* Register and Login
* Create property listings
* View public listings
* Filter properties
* Manage personal listings
* Update and delete properties
* Manage profile settings

---

## Tech Stack

Frontend:

* React
* TypeScript
* Tailwind CSS
* Axios

Backend:

* Node.js
* Express.js

Database:

* MongoDB
* Mongoose

Authentication:

* JWT
* bcrypt

---

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
